# 🎥 Video to Text UI (Base)

This is the base version of the application.

---

# ✨ Features

* Speech-to-text
* Basic UI

---

# 🚀 Setup

```bash
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app:app --reload
```

---

# ⚠️ Notes

* No TTS support in this branch
* Lightweight and stable
